# Gymmate API

## PostgreSQL Setup

Set the following environment variables (e.g., in a .env file):

```
DB_HOST=localhost
DB_PORT=5432
DB_USERNAME=postgres
DB_PASSWORD=postgres
DB_NAME=gymmate
```

## Running the Project

```
npm install
npm run start:dev
```

## Authentication & Roles

- Register: `POST /auth/register` { username, password, role? }
- Login: `POST /auth/login` { username, password }
- Roles: `admin`, `trainer`, `member`

## Example Endpoints

- Create Member (admin only): `POST /create-member` (requires Bearer token)
- List Trainers (any logged-in user): `GET /list-trainers` (requires Bearer token)
- Book Class (member only): `POST /book-class` { classId } (requires Bearer token)

Include the JWT token in the `Authorization: Bearer <token>` header for protected routes.
